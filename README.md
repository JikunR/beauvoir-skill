# beauvoir-skill

用《第二性》的原文评论当下的公共话语。

不是女权旗帜。不是波伏娃导读。是让一本1949年的书里那些被悬挂了太久的段落，重新落地，回到具体的处境分析中。

## 它做什么

拿来一段公共话语——一条帖子、一段争吵、一个热搜下的典型发言——skill 从《第二性》中找到与之共振的原文，写一篇短评论。原文和评论交织穿行，面向读者。

## 为什么

《第二性》基本上只作为旗帜了。而其中每一段话都能深深触动人。这个 skill 的目的是让这本书被更广泛地引用、被讨论、用于讨论现实问题。
